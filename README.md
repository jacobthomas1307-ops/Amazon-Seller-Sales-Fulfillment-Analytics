# Amazon Seller Sales & Fulfillment Analytics Platform

## Project Overview

This project analyzes Amazon seller transaction data to uncover business insights related to sales performance, fulfillment operations, product performance, and regional trends.

The objective is to build an end-to-end analytics solution using Python, SQL, and Power BI that enables data-driven decision-making for e-commerce operations.
